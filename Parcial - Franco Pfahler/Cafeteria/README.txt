-TUTORIAL-
Para probar las funcionalidades de la pagina web paso a paso:
* Abrir la consola
* Revisar si la ubicacion donde estas ubicado en la consola es donde se encuentra el archivo Index.HTML
* Ejecutar el comando npx serve 
* Aparecera un mensaje que dira "serving!" donde te dara el local http y la network http
* Hacer click derecho en el http de network

PERFECTO, ejecutaste con exito la pagina web!
